In 3 einfachen Schritten zur Messgeräte App

Wie funktionierts?

1. Messung am Messgerät

Ihr Messgerät sammelt alle Messdaten und sendet diese an das verbundene Industrial Gateway 4.0

2. Übermittlung an die Cloud

Das Industrial Gateway 4.0 sendet die Messdaten sicher in die Cloud

3. Verarbeitung in der Cloud

Die Cloud verarbeitet die Messdaten und stellt alle Services für die Messgeräte Anwendung bereit

4. Nutzung der App

Sie können Ihr Messgerät über die Messgeräte App bequem steuern und Messdaten auslesen
